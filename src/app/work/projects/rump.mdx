## Overview

A Unity-based side-scrolling game featuring a rainbow jumper person, designed to provide endless fun and challenges.

## Key Features

- **Endless Scrolling Gameplay**: Navigate through an infinite, procedurally generated world with increasing difficulty.
- **Dynamic Obstacles**: Jump over and avoid a variety of obstacles that test your reflexes and timing.
- **Vibrant Visuals**: Enjoy colorful, rainbow-themed graphics that create an engaging and lively experience.

## Technologies Used

- **Unity**: A powerful game engine used for creating 2D and 3D games, written in C#.
- **C#**: The primary programming language used to implement game mechanics and logic.

## Challenges and Learnings

Creating a seamless endless scrolling experience required careful attention to level design and procedural generation. Balancing the difficulty curve to keep the game engaging yet fair was another challenge. Additionally, optimizing performance for smooth gameplay on various devices was a key learning experience.
