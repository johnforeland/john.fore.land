## Overview

An old-school style isometric tower defense game featuring pixel-art graphics and engaging gameplay.

## Key Features

- **Tower Defense Gameplay**: Strategically place towers to defend against waves of enemies.
- **Pixel-Art Graphics**: Enjoy a visually appealing retro-style design with detailed sprites.

## Technologies Used

- **C++**: The primary programming language for implementing game logic and mechanics.
- **SFML**: A cross-platform library used for rendering graphics and handling input.
- **Q\***: A pathfinding algorithm used to calculate optimal enemy paths.
- **Tiled**: A map editor used for designing game levels and layouts.

## Challenges and Learnings

Developing a tower defense game required balancing gameplay difficulty and performance. Implementing the Q\* algorithm for pathfinding and optimizing it for real-time performance was a significant challenge. Additionally, creating visually appealing pixel-art and ensuring smooth animations were key learning experiences.
